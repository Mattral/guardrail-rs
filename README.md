
# 🦀 Rust in 30 Days — Beginner Roadmap

**Goal:** From zero → comfortable writing small Rust programs and understanding ownership.

---

## **Week 1: Foundations (Days 1–7)**

| Day   | Topic                                               |
| ----- | --------------------------------------------------- |
| **1** | What is Rust, setup, `main`, `println!`, variables  |
| **2** | Mutability, data types, type inference              |
| **3** | Control flow (`if`, `else`, `loop`, `while`, `for`) |
| **4** | Functions & expressions                             |
| **5** | Tuples, arrays, slices                              |
| **6** | Strings (`String` vs `&str`)                        |
| **7** | Mini practice + recap                               |

---

## **Week 2: Ownership Core (Days 8–14)**

| Day    | Topic                            |
| ------ | -------------------------------- |
| **8**  | Ownership (the big Rust concept) |
| **9**  | Borrowing & references           |
| **10** | Mutable vs immutable borrowing   |
| **11** | Lifetimes (basic intuition)      |
| **12** | Structs                          |
| **13** | Enums & `match`                  |
| **14** | Mini project (CLI toy app)       |

---

## **Week 3: Error Handling & Collections (Days 15–21)**

| Day    | Topic                          |
| ------ | ------------------------------ |
| **15** | `Option` and `Result`          |
| **16** | `panic!` vs recoverable errors |
| **17** | Vectors                        |
| **18** | HashMaps                       |
| **19** | Iterators                      |
| **20** | Closures                       |
| **21** | Practice project               |

---

## **Week 4: Practical Rust (Days 22–30)**

| Day    | Topic                        |
| ------ | ---------------------------- |
| **22** | Modules & crates             |
| **23** | File I/O                     |
| **24** | Basic CLI args               |
| **25** | Traits                       |
| **26** | Generics                     |
| **27** | Smart pointers (`Box`, `Rc`) |
| **28** | Simple concurrency           |
| **29** | Final mini project           |
| **30** | Review + next steps          |
